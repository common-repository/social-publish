directly on your facebook dashboard when you publish new post or update yours published posts

== Description ==

This plugin allows interaction with social network when specific actions in blog happen. As we do new realease we'll add more features.
It uses 

**Main Feature:**

*   Publish on facebook dashborad using secret facebook mail that you can retrieve from www.facebook.com/mobile
*   Custom message to show
*   Option when publishing

FeedBacks and suggests are welcome

== Installation ==

1. Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation.
2. Then activate the Plugin from Plugins page.
3. Go to Plugins option panel named "Social Publish" under options.
4. Setup your options.
5. Save!.
